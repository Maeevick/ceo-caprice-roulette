- Scaleway infrastructure, serverless-first
- Rust (Bevy for games)
- AI API integration (Claude, etc.)

**Constraints:**
- Solo builder — no team, no outsourcing
- Free/OSS assets only (no expensive upfront licences)
- Open-sourceable output preferred
- Legal in France/EU, ethical, no dark patterns

## SELECTION RULES FOR CLAUDE

1. **Never repeat a domain across pitches in the same batch** — diversify every session
2. **Favor 🔥🔥🔥 domains** but include at least one 🔥🔥 or 🔥 per batch of 3
3. **Rotate domains across sessions** — avoid landing on the same two domains every week
4. **Domain hints from the builder override all weights** — if they say "lean security today", all 3 can be security
5. **Wildcard flag = pick one domain the builder hasn't seen in the last 3 sessions**
6. **Never infer new domains from conversation** — if a new interest emerges, ask the builder to update this file
7. **Supporting domains** (Pixel Art, Chiptune, Interactive Fiction, Card/Puzzle Tooling) pair naturally with Game Dev and TTRPG — prefer combining their signals rather than pitching them in isolation
